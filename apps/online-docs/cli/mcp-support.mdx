---
title: "MCP support"
description: "How MCP configuration relates to Kata CLI."
---

`@kata-sh/cli` 0.16.0 does not manage MCP server configuration.

Configure MCP servers in Pi when needed. Kata CLI provides setup, backend diagnostics, and backend operation dispatch through `kata call` and `kata json`.

For backend configuration, see [CLI preferences](/reference/cli-preferences). For command details, see [Commands](/cli/commands).
