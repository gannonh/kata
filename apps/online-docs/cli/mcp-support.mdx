---
title: "MCP support"
description: "How MCP configuration relates to Kata CLI."
---

`@kata-sh/cli` 0.16.0 does not manage MCP server configuration.

Configure MCP servers in the host agent runtime when supported. Kata CLI provides setup, backend diagnostics, and runtime operation dispatch through `kata call` and `kata json`.

For backend configuration, see [CLI preferences](/reference/cli-preferences). For runtime commands, see [Commands](/cli/commands).
