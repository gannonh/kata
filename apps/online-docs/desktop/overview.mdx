title: "Kata Desktop"

Kata Desktop is the native Electron app for Kata. It runs Pi in RPC mode for agent chat and tools, and it bundles Kata CLI for the backend IO used by Kata skills. Desktop adds a visual interface for session management, workspace switching, provider setup, workflow kanban, and Symphony operation.

## Features

    Point Desktop at any local workspace and keep sessions, workflow state, and Symphony config scoped to that project.

## Installation

Download a packaged build from [GitHub Releases](https://github.com/gannonh/kata/releases) or build from source.

### Build from source

From the repository root:

```bash
pnpm install
pnpm --dir apps/desktop run desktop:dev
```

For a packaged macOS `.dmg`:

```bash
pnpm --dir apps/desktop run desktop:dist:mac
```

## Tech stack

| Layer | Technology |
|---|---|
| Runtime | Electron with a Node.js main process and Chromium renderer |
| Agent execution | Pi RPC bridge with bundled Kata skills |
| Backend IO | Kata CLI |
| UI | React 19, Tailwind CSS v4, Radix UI, Jotai |
| Build | esbuild, Vite, electron-builder |
| Auth | `~/.kata-cli/agent/auth.json` |
| Sessions | `~/.kata-cli/sessions/*.jsonl` |
