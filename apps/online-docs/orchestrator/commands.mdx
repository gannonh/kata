<Warning>
  Kata Orchestrator commands are archived with `apps/orchestrator-legacy` and are not part of the active Kata CLI 0.16.0 command surface.
</Warning>

Use current Kata CLI commands:

```bash
kata setup
kata doctor
kata call <operation> --input <request.json>
kata json <request.json>
```

See [CLI commands](/cli/commands) for the active command surface.
