# Symphony

## Overview

Symphony (`@kata/symphony`) is a Rust binary that coordinates multiple agents across tasks. It handles scheduling, resource allocation, and inter-agent communication.

## Architecture

Symphony compiles to a standalone binary. It communicates with the rest of the monorepo through a well-defined IPC interface.

## Usage

Symphony is invoked through the CLI or directly as a binary. See the monorepo's `apps/symphony/` directory for build instructions.
