# Orchestrator

## Overview

The Orchestrator (`@kata-sh/orc`) is a meta-prompting system that composes agent behaviors from modular building blocks. It generates system prompts, tool configurations, and workflow definitions.

## Architecture

The orchestrator assembles prompts from fragments, skills, and context. It runs as a build-time step that produces the final agent configuration.

## Usage

The orchestrator is invoked during CLI startup to assemble the active agent configuration. It can also be used standalone for prompt development.
