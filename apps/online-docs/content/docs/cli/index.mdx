# CLI

## Key Features

The Kata CLI (`@kata-sh/cli`) provides an interactive agent that runs in your terminal. It supports multiple workflow modes, extensible tool use, and configurable agent behaviors.

## Getting Started

Install the CLI and run your first session:

```bash
bun install -g @kata-sh/cli
kata
```

See [Installation](/docs/getting-started/installation) and [Quickstart](/docs/getting-started/quickstart) for detailed setup instructions.
