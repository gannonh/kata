# Kata Workflow

## Step Mode

Step mode pauses after each action for your review. Use this when you want full control over what the agent does.

## Auto Mode

Auto mode lets the agent execute a plan end-to-end without pausing. Best for well-defined tasks where you trust the agent's judgment.

## Discuss Mode

Discuss mode is a conversational interface. The agent answers questions and proposes changes but does not execute them without confirmation.
