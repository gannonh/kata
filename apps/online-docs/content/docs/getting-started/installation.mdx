# Installation

## Prerequisites

You need Node.js 18+ and Bun 1.3.8+ installed. Rust is required only if you plan to work on Symphony.

## Install

Install the CLI globally:

```bash
bun install -g @kata-sh/cli
```

## Verify

Confirm the installation succeeded:

```bash
kata --version
```
