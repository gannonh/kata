# Quickstart

## Your First Session

Launch an interactive session with the CLI:

```bash
kata
```

The agent will start in discuss mode, ready to answer questions about your codebase.

## Next Steps

- Explore [CLI commands](/docs/cli/commands) for the full command reference.
