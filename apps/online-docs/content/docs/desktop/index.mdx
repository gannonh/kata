# Desktop & Viewer

## Desktop App

The Kata Desktop app is an Electron application that provides a graphical interface for interacting with the agent. It supports the same workflow modes as the CLI.

## Session Viewer

The session viewer (`apps/viewer`) is a Vite-based web app for reviewing past agent sessions. It renders conversation transcripts with full tool-call detail.

## Development

Run the desktop app in dev mode:

```bash
bun run electron:dev
```
