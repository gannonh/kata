# Conventions

## TypeScript

All JavaScript/TypeScript code uses strict TypeScript. Shared types live in `@craft-agent/core`. Prefer explicit types over inference at package boundaries.

## Testing

Turborepo orchestrates all test runners. Most packages use Bun test. Context uses Vitest (native addon requirement). Symphony uses `cargo test`. Pre-push hooks run `turbo run lint typecheck test --affected`.

## Git Workflow

The repo uses git worktrees for parallel development. Each worktree has a standby branch tracking main. Feature branches are created from standby branches. PRs target main.
