# Packages

## @craft-agent/core

Core TypeScript types shared across all apps and packages. Contains interface definitions, enums, and type utilities.

## @craft-agent/shared

Business logic shared between CLI and Desktop. Includes agent runtime, auth, config management, MCP integration, channels, and daemon communication.

## @craft-agent/ui

Shared React components for chat interfaces and markdown rendering. Used by both the Desktop app and the Viewer.

## @craft-agent/mermaid

Mermaid diagram-to-SVG renderer. Converts Mermaid syntax into static SVG images for embedding in agent responses.
