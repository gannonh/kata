# Phase Plan Artifact Template

Use this as the content shape for slice-scoped `plan` artifacts.

```markdown
# Plan: [Slice Title]

## Goal

[What this slice delivers.]

## Requirements

- [REQ-01]

## Must Haves

- [Observable outcome that must be true]

## Tasks

1. [Task title]
   - Description: [what to do]
   - Verification: [how to know it works]

## Risks

- [Risk and mitigation]

## Files / Areas Likely Touched

- [Path or subsystem]
```

Plans must be executable by a fresh agent without hidden context.
