# Summary Artifact Template

Use this as the content shape for `summary` artifacts.

```markdown
# Summary: [Scope]

## What Changed

- [Change]

## Requirements Completed

- [REQ-01]

## Verification

- [Command or manual check]

## Commit

- [Commit hash, or "No code commit required" with reason]

## Follow-Ups

- [Follow-up]
```
